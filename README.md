## Try it out

## Useful cmd
- Uninstall app: `adb uninstall "com.tozaicevas.guess_the_place"`
- Cleanup gradle: `./gradlew clean`
- Make a release build: `./gradlew bundleRelease`
- Run release version `npx react-native run-android --variant=release`
